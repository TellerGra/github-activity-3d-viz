# GitHub Activity 3D Visualization

## Project Documentation

The GitHub Activity 3D Visualization project visualizes GitHub user activity in a 3D space. It takes user data from GitHub and transforms it into an interactive 3D visualization.

## Usage Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/TellerGra/github-activity-3d-viz.git
   cd github-activity-3d-viz
   ```

2. **Install Dependencies**
   Make sure you have all the required libraries installed. Use the following command:
   ```bash
   npm install
   ```

3. **Run the Project**
   To start the visualization, use the command:
   ```bash
   npm start
   ```

4. **Access the Visualization**
   Open your web browser and navigate to `http://localhost:3000` to view the 3D visualization of GitHub activity.
